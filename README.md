# Asoftware - Instaladores

Punto de descarga oficial de **Asoftware**, software de presupuestos y gestion para carpinteria de aluminio.

## Descargar la ultima version

Descargá el instalador mas reciente desde **[Releases](https://github.com/germanroz/Asoftware-releases/releases/latest)**.

> Solo la ultima version queda disponible para descargar. Los releases anteriores se conservan como historial de cambios, pero sin instalador adjunto.

## Requisitos

- Windows 10 (build 10240) o superior, 64 bits.
- .NET 8 Desktop Runtime (el instalador avisa si falta).

---

*El codigo fuente es privado. Este repositorio contiene unicamente los instaladores distribuibles.*
